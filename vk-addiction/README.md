# vk-addiction
**Цели проекта:** оценка степени зависимости подростка от социальной сети ВКонтакте.  

**Решение:** 
 - [Веб-скрейпер](https://github.com/zaikinalex75/da-ds-portfolio/blob/main/vk-addiction/vk-15.py) - раз в 15 минут запрашивает страницу подростка в ВК, считывает с неё статус и выводит его на экран (вывод экрана перенаправляется в [лог-файл](https://github.com/zaikinalex75/da-ds-portfolio/blob/main/vk-addiction/portfolio-vk-log.txt)).
 - [тетрадь Jupyter Notebook](https://github.com/zaikinalex75/da-ds-portfolio/blob/main/vk-addiction/portfolio-vk.ipynb) - загрузка лог-файла, предобработка (пропуски, сбои), исследование и визуализация посещений ВК в разрезе дней, часов и недель.

**Темы:** web scraping, регулярные выражения, предобработка данных, визуализация и исследование временных рядов.

**Стек:** python, requests, pandas, matplotlib.
